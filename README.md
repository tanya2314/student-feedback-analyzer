Student Feedback Analyzer
This is a simple Python-based project that collects and analyzes student feedback.

## Features
- Add student feedback with ratings
- Calculate average rating
- Display overall performance

## How It Works
The program takes input from users, stores feedback, and calculates the average rating to determine performance.

## Technologies Used
- Python

## Purpose
This project helped me understand basic programming concepts such as input handling, loops, conditions, and simple data analysis.
